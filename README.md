# Discipline Hub

A React, Tailwind CSS, and Framer Motion life gamification app for habit tracking, discipline analytics, goals, and visual self-improvement progress.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Publish with GitHub Pages

This repo includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.

After pushing to a GitHub repository:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to the `main` branch.

GitHub will build the app and publish the `dist` folder automatically.
